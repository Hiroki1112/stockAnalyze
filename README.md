# stockAnalyzer

Kabu+からのデータを分析するためのプログラム。
関数はできるだけ小さく作っています。

func.py -> 関数群をまとめたファイル。
main.py ->　関数を実行するファイル。

ファイル構成
``` bash
stock ┳ code　┳　main.py
      ┃       ┗　func.py
      ┃
      ┃
      ┣ daily 
      ┃
      ┣ stockdata 
      ┃
      ┣ output 
      ┃
      ┗ tmp
```
